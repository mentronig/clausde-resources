Dokumentierter Workflow unter `docs/workflow-anfrage.md` seit 2026-04-03:

1. **Anfrage ablegen** – inbox → auftraege/YYYY-MM_kunde-projekt/anfrage/
2. **Anfrage analysieren** – Strukturierte Auswertung → analyse/anfrage-analyse.md
3. **Kurzprofil erstellen** – Template-Check (Vermittler?), dann zuschneiden → angebot/
4. **Profil-Anfrage-Abgleich** – Kritische Checkliste → analyse/profil-abgleich.md

**Why:** Roland braucht einen wiederholbaren Prozess für eingehende Kundenanfragen.

**How to apply:** Bei jeder neuen Anfrage in der inbox diesen Workflow befolgen. Bei Annahmen immer erst Roland fragen.
