Rolands "Second Brain" ist ein Obsidian Vault unter `C:\opt\Projects\obsidian-ws\Mentronig`.

Struktur (PARA-ähnlich):
- **00 Kontext/** – Persönliches Profil, ICP, Angebot, Schreibstil, Branding
- **01 Inbox/** – Unverarbeitete Notizen
- **02 Projekte/** – Aktive Projekte mit Enddatum
- **03 Bereiche/** – Laufende Verantwortungsbereiche
- **04 Ressourcen/** – Referenzmaterial zu KI und Agilität
- **05 Daily Notes/** – Tägliches Logbuch (YYYY-MM-DD.md)
- **06 Archiv/** – Abgeschlossene Projekte
- **07 Anhänge/** – Medien/PDFs

Die CLAUDE.md im Vault definiert Session-Routinen (Inbox prüfen bei Start, Daily Note bei Ende).
